# Anastamar Dental Clinic Website

This project is now structured as a **Next.js + Tailwind CSS** application and is ready for Vercel deployment.

## Local development

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
npm run start
```

## Deploy to Vercel

1. Push this repository to GitHub/GitLab/Bitbucket.
2. Import the repository in [Vercel](https://vercel.com/new).
3. Vercel will automatically detect it as a Next.js project.
4. Keep default settings and click **Deploy**.

No extra Vercel configuration file is required.
